# blu-board

A custom devboard (my first!) Built off of a RP2040, this is a more advanced project so I could get used to complex routing, placement, and assembly. I started this off using a guide, which I consulted for fabrication, materials, and for when I ran into DRC errors I'm not familair with. I decided to customize it and add a special LED that will glow periodically. I made it with the intention of creating a PCB that would be my most diificult to route yet in order to challenge myself, and it definetly was!
